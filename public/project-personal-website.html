Personal Portfolio Website Development

This project involved the development of a professional-quality personal portfolio website to effectively showcase my skills, experiences, and projects. The website was constructed using HTML, CSS, and Bootstrap for frontend development and deployed on Firebase Hosting

• Frontend Development: HTML was utilized to establish the website's structure, while CSS was employed to define its visual style and layout. Bootstrap's responsive framework was integrated to ensure optimal presentation across various devices

• Deployment: Firebase Hosting provided the platform for deploying the live website, guaranteeing accessibility and performance
